# AI 知识库

本目录存放用于 RAG（检索增强生成）的原始知识文档。

## 目录结构

```
knowledge-base/
├── courses/        # 课程信息文档
├── jobs/           # 招聘岗位与行业知识
├── interviews/     # 面试题库与技巧
└── policies/       # 学校政策文件
```

## 文档格式

支持 `.txt` 和 `.md` 格式。

## 使用方法

1. 将文档放入对应分类目录
2. 通过 AI 智能体管理后台的"知识库"页面上传
3. 系统自动完成向量化处理
4. AI 对话和业务场景会基于知识库内容进行检索增强
