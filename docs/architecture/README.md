## 技术选型理由

| 组件 | 选型 | 理由 |
|------|------|------|
| 后端框架 | Spring Boot 3.2 | 成熟稳定，生态完善，JDK21+虚拟线程 |
| ORM | MyBatis Plus 3.5 | 简化CRUD，Lambda查询，分页插件 |
| 认证 | Spring Security + JWT | 无状态认证，适合前后端分离 |
| 实时通信 | WebSocket | 学业预警实时推送 |
| 缓存 | Redis | Token黑名单，热点数据缓存 |
| 管理后台 | Vue 3 + Element Plus | 组件丰富，开发效率高 |
| 小程序 | 原生框架 | 性能最优，无额外依赖 |

## 数据库设计

采用关系型数据库 MySQL，核心实体关系：

```
sys_user ──┬── academic_student ──┬── academic_grade
           │                      ├── academic_warning
           │                      ├── academic_study_plan
           │                      ├── employ_resume ─── employ_job_match
           │                      └── employ_interview
           ├── ai_conversation
           └── sys_user_role ── sys_role ── sys_role_menu ── sys_menu
```

## 安全设计

- 密码 BCrypt 加密存储
- JWT Token 无状态认证（24小时过期）
- 接口基于角色权限控制（ADMIN / TEACHER / STUDENT）
- CORS 跨域配置
- 敏感配置环境变量注入
