# 用户指南

## 学生端（微信小程序）

### 首页
展示学业概览、最新预警和推荐岗位。

### 学业情况
- 查看 GPA、学分进度、不及格科目
- 接收学业预警通知
- 查看个性化学习规划

### 我的简历
- 查看 / 编辑个人简历
- 使用 AI 生成和优化简历
- 设定目标岗位

### 职位搜索
- 搜索和筛选招聘岗位
- 查看岗位详情
- 一键投递简历

### AI 模拟面试
- 选择文字或语音面试模式
- 与 AI 面试官实时对话
- 获取面试评分与反馈

---

## 教师/管理员端（PC 管理后台）

访问地址：`http://localhost:3000`

### 登录
默认账号：admin / admin123

### 学业管理
- **学生管理**：查看全校学生学业数据
- **学业预警**：查看预警列表，处理预警并添加备注

### 就业服务
- **简历管理**：查看学生简历，使用 AI 生成简历
- **岗位管理**：发布/编辑/下架招聘岗位
- **面试记录**：查看学生模拟面试记录

### AI 智能体
- **知识库**：上传课程、岗位、面试题等文档，丰富 AI 回答

### 系统管理
- **用户管理**：管理学生、教师、管理员账号
- **角色管理**：查看系统角色配置
